The SVG foreignObject attribute **systemLanguage** can be used to detect the OS language and display [this example page](https://mardeg.github.io) in that language, defaulting to English if the detected language doesn't match.

The languages are:

* en (English, default)
* fr (French)
* de (German)
* ru (Russian)
* it (Italian)
* nl (Dutch)
* es (Spanish)
* ja (Japanese)
* sv (Swedish)

and Indian languages:
* hi (Hindi)
* bn (Bengali)
* ta (Tamil)
* te (Telugu)
* gu (Gujarati)
* pa (Punjabi)
* ur (Urdu)
* or (Odia)
* mr (Marathi)
